# 🧮 Data Structure - Circular Queue with C

> Data Structure เรื่อง Circular Queue แบบ Array ด้วยภาษา C

## 🛠️ Features
- **Enqueue**: เพิ่มข้อมูลที่ด้านท้ายของคิว
- **Dequeue**: ลบข้อมูลที่ด้านหน้าของคิว
- **Display**: แสดงข้อมูลทั้งหมดในคิว
